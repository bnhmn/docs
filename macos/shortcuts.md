# macOS Shortcuts

This is a list of useful keyboard shortcuts for macOS.

```text
Windows-Key = Command ⌘
Alt-Key = Option ⌥
Ctrl-Key = ⌃
```

## macOS

```text
Make Screenshot:                              Shift-Cmd-4
Refresh:                                      Cmd-R
Lock screen:                                  Ctrl-Cmd-Q
Search Apps (Spotlight):                      Cmd-Space
Open Emoji keyboard:                          Ctrl-Cmd-Space
Mute MS Teams:                                Cmd-Shift-M
Show Desktop                                  F11
Open Browser Developer Console:               F12
Ctrl+Space default:                           Switch input sources -> Disable under
                                              Prefs>Keyboard>Shortcuts>Input Sources

Finder Show hidden files:                     Cmd-Shift-.
Finder Go To Folder:                          Cmd-Shift-G
Finder New Folder:                            Cmd-Shift-N
Finder Copy Path:                             Right Click File + Press Alt
Finder Change default app for extension:      Right click file in Finder
                                              -> Information
                                              -> Name & Suffix
                                              -> Change App and click "Change for all"

Open iOS Simulator                            open -a Simulator.app
Install unsigned DMG/PKG App:                 xattr -d com.apple.quarantine <NAME>.pkg
```
